## 核心组件

### 1. Poisson Bracket 算符
**文件**: `src/pytokmhd/operators/poisson_bracket.py`

#### 正则Poisson括号

```python
[f, g] = (1/R²) (∂f/∂r ∂g/∂θ - ∂f/∂θ ∂g/∂r)
```

其中 R = R₀ + r·cos(θ) 是大半径。

#### 数学性质（已验证）

| 性质 | 公式 | 验证精度 |
|-----|------|---------|
| **反对称性** | [f, g] = -[g, f] | 1e-14 (机器精度) |
| **线性** | [af+bg, h] = a[f,h] + b[g,h] | 1e-12 |
| **Jacobi恒等式** | [f,[g,h]] + [g,[h,f]] + [h,[f,g]] = 0 | 3e-3 (2阶精度) |

#### 实现细节

- **离散化**: 2阶中心差分
- **边界条件**:
  - θ方向: 周期边界
  - r方向: 单侧2阶差分
- **辅助函数**:
  - `jacobi_identity_residual()`: 测试Jacobi恒等式
  - `advection_bracket()`: MHD对流项的物理命名

### 2. Hamiltonian 能量泛函
**文件**: `src/pytokmhd/physics/hamiltonian.py`

#### 总Hamiltonian

```python
H[ψ, φ] = ∫ d³x [(1/2)|∇φ|² + (1/2)|∇ψ|²]
         = K + U
```

其中:
- **K = ∫(1/2)|∇φ|² dV**: 动能（E×B流）
- **U = ∫(1/2)|∇ψ|² dV**: 磁能（极向场）

#### 体积元

环形坐标系:
```
d³x = √g dr dθ dφ = r·R dr dθ dφ
```

轴对称情况（∂/∂φ = 0）:
```
∫₀²ᵖ dφ = 2π
```

#### 能量密度

```python
h = (1/2)[(∂φ/∂r)² + (1/r²)(∂φ/∂θ)²] 
  + (1/2)[(∂ψ/∂r)² + (1/r²)(∂ψ/∂θ)²]
```

#### 数值积分

使用梯形法则:
```python
H = 2π ∫∫ h · (r·R) dr dθ
  ≈ 2π Σᵢⱼ h[i,j] · J[i,j] · dr · dθ
```

## 物理验证

### 能量守恒结构

演化方程:
```
∂ψ/∂t = [ψ, H]
∂ω/∂t = [ω, H]  其中 ω = ∇²φ
```

理论保证:
```
dH/dt = ∫ [δH/δψ · ∂ψ/∂t + δH/δφ · ∂φ/∂t] dV
      = ∫ [δH/δψ · [ψ,H] + δH/δφ · [φ,H]] dV
      = 0  (反对称性)
```

### 数值收敛性

测试显示Jacobi恒等式残差随网格细化减小:
- 粗网格 (32×64): residual = 1.13e-2
- 细网格 (64×128): residual = 2.99e-3

符合2阶精度预期: residual ~ O(dr² + dθ²)

## 实现决策

### 1. 度规处理

**问题**: 环形坐标梯度涉及复杂的度规因子。

**决策**: 从原始导数 ∂/∂r, ∂/∂θ 直接计算 |∇f|²，避免与 `gradient_toroidal()` 的物理分量混淆。

**代码**:
```python
from ..operators.poisson_bracket import _compute_derivatives

dpsi_dr, dpsi_dtheta = _compute_derivatives(psi, grid)
grad_psi_squared = dpsi_dr**2 + (dpsi_dtheta / r_grid)**2
```

### 2. Jacobi恒等式阈值

**问题**: 离散化误差导致Jacobi恒等式不精确满足。

**决策**: 设置阈值 1e-2（比初始1e-6宽松），与2阶精度O(dr²+dθ²)一致。

**验证**: 收敛测试确认误差随网格细化减小。

### 3. 能量积分

**问题**: 环形几何体积积分需要Jacobian。

**决策**: 使用 `grid.jacobian()` (即 √g = r·R)，在极向平面积分后乘 2π。

**物理**: 轴对称假设自然地简化了φ方向的积分。

### Hamiltonian Energy

```python

    compute_hamiltonian,
    hamiltonian_density,
    kinetic_energy,
    magnetic_energy,
    energy_partition,
)

h = hamiltonian_density(psi, phi, grid)      # 能量密度场
```
