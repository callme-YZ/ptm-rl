## v1.2: Toroidal Dissipative MHD + 2D RL

**Goal:** Establish stable toroidal solver + IMEX framework + RL baseline

**Physics:**
```
∂ψ/∂t = [ψ,φ]_toroidal + η∇²ψ
∂ω/∂t = [ω,φ]_toroidal + ν∇²ω
η, ν ~ O(10⁻⁴) realistic
```

**Integrator: IMEX**
- Explicit (RK2): Poisson bracket [ψ,φ] (Arakawa scheme)
- Implicit (BE): Diffusion η∇²ψ, ν∇²ω

**RL:**
- Action: 2D boundary [a₀, a₁] ∈ [-1,1]²
- Observation: psi_modes(8) + omega_modes(8) + E(1) + dE/dt(1) = 18D
- Reward: -island_width (single-objective)

**Deliverable:**
1. Toroidal solver stable (t>1000)
2. IMEX validated
3. 2D RL learns suppression >20% vs no-control

**Publication:** Physics of Plasmas (6-7/10)

## v1.3: Scalar Elsässer Transformation

**Goal:** Learn Elsässer formulation + Cross-helicity dynamics

**Physics:**
```
z⁺ = ω + ψ, z⁻ = ω - ψ (scalar Elsässer)
∂z⁺/∂t = [z⁺,φ⁺] + η∇²z⁺
∂z⁻/∂t = [z⁻,φ⁻] + ν∇²z⁻
H_C = ∫z⁺·z⁻ dV (cross-helicity)
```

**Integrator:** Same IMEX, for (z⁺,z⁻)

**RL:**
- Action: 2D boundary (same as v1.2)
- Observation: z⁺_modes(8) + z⁻_modes(8) + H_C(1) + dH_C/dt(1) = 18D
- Reward: -island_width

**Deliverable:**
1. Elsässer formulation working
2. H_C conservation validated (ideal limit)
3. RL policy transfer from v1.2

**Publication:** Physics of Plasmas (7/10)

## v1.4: Spatial Control + Multi-Objective

**Goal:** Realistic RL control with spatial J_ext

**Physics:**
```
z⁺, z⁻ (scalar) + J_ext
J_ext = Σ aᵢ * Gaussian_i(r,θ)
6 Gaussian bumps (3 radial × 2 poloidal)
```

**RL:**
- Action: [a₁,...,a₆] ∈ [-1,1]⁶ (Doc 3 design)
- Observation: z⁺_modes(8) + z⁻_modes(8) + E(1) + drift(1) + H_C(1) = 19D
- Reward: Multi-objective (Doc 4 design)
  - r = -w_island*island_width - w_power*(P/P_max) - w_constraint*exp(div_B)

**Multi-objective strategy:**
- Train 3 policies (performance/balanced/efficiency)
- Pareto front exploration

**Deliverable:**
1. Spatial J_ext control
2. Multi-objective RL
3. Beat 4 baselines in 2/3 metrics

**Publication:** Physical Review Letters (8/10)

## v2.0: Full 3D Vector Elsässer

**Goal:** Production system (YZ vision 达成)

**Physics:**
```
z⁺ = u + B, z⁻ = u - B (true 3D Elsässer)
H = ∫(|z⁺|²+|z⁻|²)/4 dV
H_C = ∫z⁺·z⁻ dV
```

**RL:**
- Action: 3D Elsässer forcing f⁺,f⁻ or 3D J_ext (12-18D)
- Observation: ~30-40D
- Multi-objective: tearing + turbulence + confinement

**Deliverable:**
1. Production Hamiltonian MHD-RL
2. 足够好物理 + 保结构 + RL探索
3. External review 9+/10

**Publication:** Nature Physics / PRX (9-10/10)
