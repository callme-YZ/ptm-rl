## Part 1: Toroidal Geometry Upgrade

### Step 1.1: Coordinate System Design

**Current (v1.0):**
```python
# Cylindrical coordinates
r: radial (minor radius)
z: vertical (poloidal)
φ: (ignored, axisymmetric assumption)
```

**Target (v1.1):**
```python
# Toroidal coordinates
R: major radius (R = R₀ + r*cos(θ))
φ: toroidal angle
Z: vertical (Z = r*sin(θ))

where:
  R₀ = major radius (e.g., 1.0 m)
  r = minor radius
  θ = poloidal angle
```

**Metric Tensor:**
```
g₁₁ = 1           (∂r/∂r)
g₂₂ = r²          (∂θ/∂θ)
g₃₃ = R²          (∂φ/∂φ)
√g = r*R          (Jacobian)
```

**Implementation:**
- Define `ToroidalGrid` class
- Compute metric tensor components
- Implement coordinate transformation utilities

**Validation Criteria:**
- ✅ Metric tensor satisfies orthogonality: g_ij = 0 for i≠j
- ✅ Jacobian √g > 0 everywhere
- ✅ Coordinate transformation invertible (r,θ,φ ↔ R,Z,φ)

### Step 1.2: Differential Operators in Toroidal Geometry

**Gradient:**
```
∇f = (1/√g) * [∂f/∂r, (1/r²)∂f/∂θ, (1/R²)∂f/∂φ]
```

**Divergence:**
```
∇·A = (1/√g) * [∂(√g A_r)/∂r + ∂(√g A_θ)/∂θ + ∂(√g A_φ)/∂φ]
```

**Laplacian:**
```
∇²f = (1/√g) * [∂/∂r(√g/g₁₁ ∂f/∂r) + ∂/∂θ(√g/g₂₂ ∂f/∂θ) + ∂/∂φ(√g/g₃₃ ∂f/∂φ)]
```

**Simplified (axisymmetric ∂/∂φ = 0):**
```
∇²f = (1/r*R) * [∂/∂r(r*R ∂f/∂r) + ∂/∂θ(R/r ∂f/∂θ)]
```

**Implementation:**
- Implement `gradient_toroidal(f, grid)`
- Implement `divergence_toroidal(A, grid)`
- Implement `laplacian_toroidal(f, grid)`

**Validation Criteria:**
- ✅ ∇·(∇f) = ∇²f (identity check)
- ✅ Laplacian of constant = 0
- ✅ Test on known analytical function (e.g., f = R² + Z²)
- ✅ Numerical precision: error < 1e-12 (machine precision)

## Part 2: Symplectic Time Integration

### Step 2.1: Symplectic Integrator Selection

**Current (v1.0):**
```python
# 4th-order Runge-Kutta (RK4)
# Non-symplectic, energy drift O(1) over long time
```

**Target (v1.1):**
```python
# Symplectic integrator (choose one):
Option A: Störmer-Verlet (2nd-order, symplectic)
Option B: 4th-order symplectic (Forest-Ruth)
Option C: Implicit midpoint (2nd-order, symplectic)
```

**小P推荐:** **Störmer-Verlet** (simple, proven, 2nd-order sufficient)

**Why Symplectic?**
- Preserves phase-space volume (Liouville's theorem)
- Long-time energy drift: O(dt²) per step, O(1) total (bounded)
- Standard RK4: energy drift unbounded, grows linearly

**Mathematical Background:**
For Hamiltonian system dq/dt = ∂H/∂p, dp/dt = -∂H/∂q:
```
Symplectic condition: J^T M J = J
where J = [[0, I], [-I, 0]] (symplectic matrix)
```

**Implementation:**
- Implement `StormerVerlet` class
- Interface: same as `RK4Integrator`
- Split MHD into kinetic + potential parts (if possible)

**Validation Criteria:**
- ✅ Energy drift < 1e-6 over 10⁴ steps
- ✅ Phase-space volume conserved (test with ensemble)
- ✅ Reversibility: forward then backward returns to initial state (error < 1e-10)

**Challenge:** Reduced MHD is not obviously Hamiltonian

**Approach 1: Splitting (if possible)**

**Validation Criteria:**

- ✅ Long-time stability (>10⁴ steps without divergence)
- ✅ Symplectic structure approximately preserved

### Step 2.3: Symplectic Integrator Implementation

**Störmer-Verlet Algorithm:**
```python
def stormer_verlet_step(psi, omega, dt, compute_rhs):
    # Step 1: Half-step omega
    dpsi_dt, domega_dt = compute_rhs(psi, omega)
    omega_half = omega + 0.5 * dt * domega_dt
    
    # Step 2: Full-step psi
    dpsi_dt, _ = compute_rhs(psi, omega_half)
    psi_new = psi + dt * dpsi_dt
    
    # Step 3: Half-step omega
    _, domega_dt = compute_rhs(psi_new, omega_half)
    omega_new = omega_half + 0.5 * dt * domega_dt
    
    return psi_new, omega_new
```

**Note:** This is simplified; actual MHD may need implicit treatment for stiffness.

**Implementation:**
- Create `SymplecticIntegrator` class
- Support both Störmer-Verlet and (optional) 4th-order symplectic
- Maintain same interface as `RK4Integrator` for drop-in replacement

**Validation Criteria:**
- ✅ Interface compatibility: can replace RK4 with no code change
- ✅ Stability for same dt as RK4
- ✅ Energy conservation better than RK4 (quantitative test)
