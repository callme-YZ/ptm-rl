## Improvement 3: Stricter Energy Conservation Tests

### Current Status (v1.2)

**Criteria:**
- Equilibrium: 14% drift (1000 steps)
- Perturbed: 15% drift (100 steps)

**Thresholds:**
- < 20% (equilibrium)
- < 25% (perturbed)

**Issue:** Too relaxed, includes toroidal transient

### v1.3 Solution

**With proper equilibrium:**
```python
def test_strict_energy_conservation():
    """
    With true GS equilibrium, expect < 1% drift.
    """
    # Use Solovev equilibrium
    psi_eq = solovev_equilibrium(grid, beta, q0)
    omega_eq = laplacian_toroidal(psi_eq, grid)
    
    # Should be exact equilibrium
    E_initial = compute_energy(psi_eq, omega_eq)
    
    # Run 1000 steps
    for _ in range(1000):
        solver.step()
    
    E_final = solver.compute_energy()
    drift = abs(E_final - E_initial) / E_initial
    
    assert drift < 0.01  # < 1% (strict)
```
