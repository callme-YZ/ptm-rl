## Improvement 3: Toroidal Poisson Solver

### Current Status (v1.2)

**Solver:** FFT-based simple Poisson  
**Location:** `src/pytokmhd/solver/poisson_solver.py`

**Issues:**
- Uses cylindrical approximation (R=constant)
- Residual ~4-60 depending on geometry
- Not consistent with `laplacian_toroidal` operator

**Impact:**
- Geometric error O(ε) where ε=a/R₀
- For typical tokamak (ε~0.3): ~30% inaccuracy in worst cases
- Affects vorticity dynamics accuracy

### v1.3 Solution ✅ IMPLEMENTED (2026-03-19)

**Toroidal Iterative Poisson Solver**

**Method:**
- GMRES iterative solver
- Uses PyTokMHD `laplacian_toroidal` as operator
- Correct boundary condition enforcement via identity rows

**Key Learning:** BC must be identity rows in operator matrix, not projection after solve

**Performance:**
- Interior residual: **8e-8** (vs 4-60 for FFT)
- Boundary condition error: **1e-7**  
- GMRES iterations: ~100-200 (32×64 grid)
- Cost: ~10× FFT, still O(0.1s) per solve

**Note:** φ=r² is cylindrical solution; in toroidal geometry ∇²(r²) varies by ~33% due to R(θ) dependence. Solver correctly solves the toroidal equation to 8e-8 precision.

#### 3.2 Poisson Bracket [ψ, φ] (MUST-HAVE)

**Physics:**
```python
[f, g] = ∂f/∂r · ∂g/∂θ - ∂f/∂θ · ∂g/∂r
```

**Role in MHD:**
- Nonlinear advection: E×B drift
- Couples flux ψ and stream function φ
- **Essential for equilibrium maintenance**

**Why Solovev needs it:**
- Solovev equilibrium: J×B = ∇P balanced dynamically
- [ψ, φ] term transports energy/momentum
- Without it: diffusion breaks balance → explosion

**v1.3 Implementation:**
```python
class PoissonBracket:
    def __init__(self, grid):
        self.grad = ToroidalGradient(grid)
    
    def bracket(self, f, g):
        """
        Compute [f, g] = ∂f/∂r · ∂g/∂θ - ∂f/∂θ · ∂g/∂r
        """
        df_dr, df_dtheta = self.grad(f)
        dg_dr, dg_dtheta = self.grad(g)
        
        return df_dr * dg_dtheta - df_dtheta * dg_dr
```

#### 3.3 Pressure Gradient ∇P (MUST-HAVE)

**Physics:**
Force balance requires:
```python
J × B = ∇P
```

**Current v1.2:**
- No pressure term
- Assumes P ≈ 0 (cold plasma)
- **Fails for finite-β equilibria**

**v1.3 Requirement:**
```python
def pressure_profile(psi, beta_p=0.1):
    """
    Pressure as function of flux: P = P(ψ)
    
    For Solovev: P = β_p · ψ² (polynomial)
    """
    return beta_p * psi**2

def pressure_gradient(psi, grid, beta_p):
    """
    Compute ∇P = dP/dψ · ∇ψ
    """
    dP_dpsi = 2 * beta_p * psi
    grad_psi = gradient_toroidal(psi, grid)
    return dP_dpsi * grad_psi
```

**Integration into RHS:**
```python
# Vorticity equation (force balance):
dpsi_dt = [psi, phi] - eta * J
domega_dt = [omega, phi] - div(J×B - ∇P) - nu * laplacian(omega)
#                            ^^^^^^^^^^^
#                            Force balance term
```

**Effort:** ~2-4 days
