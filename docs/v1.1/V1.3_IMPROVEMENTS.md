#### 3.2 Poisson Bracket [Ōą, ŌÜ] (MUST-HAVE)

**Physics:**
```python
[f, g] = ‚ąāf/‚ąār ¬∑ ‚ąāg/‚ąāőł - ‚ąāf/‚ąāőł ¬∑ ‚ąāg/‚ąār
```

**Role in MHD:**
- Nonlinear advection: E√óB drift
- Couples flux Ōą and stream function ŌÜ
- **Essential for equilibrium maintenance**

**Why Solovev needs it:**
- Solovev equilibrium: J√óB = ‚ąáP balanced dynamically
- [Ōą, ŌÜ] term transports energy/momentum
- Without it: diffusion breaks balance ‚Üí explosion

**v1.3 Implementation:**
```python
class PoissonBracket:
    def __init__(self, grid):
        self.grad = ToroidalGradient(grid)
    
    def bracket(self, f, g):
        """
        Compute [f, g] = ‚ąāf/‚ąār ¬∑ ‚ąāg/‚ąāőł - ‚ąāf/‚ąāőł ¬∑ ‚ąāg/‚ąār
        """
        df_dr, df_dtheta = self.grad(f)
        dg_dr, dg_dtheta = self.grad(g)
        
        return df_dr * dg_dtheta - df_dtheta * dg_dr
```
