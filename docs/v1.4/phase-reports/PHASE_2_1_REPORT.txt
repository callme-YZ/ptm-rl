================================================================================
IMPLEMENTATION APPROACH
================================================================================

1. Gradient Operators:
   - ∂/∂r: 2nd-order centered FD (1st-order at boundaries)
   - ∂/∂θ: 2nd-order centered FD (periodic BC)
   - ∂/∂ζ: FFT spectral derivative (reused from Phase 1.1)

2. Metric Factor:
   - |∇ψ|² = (∂ψ/∂r)² + (1/r²)(∂ψ/∂θ)² + (∂ψ/∂ζ)²
   - 1/r² from cylindrical metric tensor g^θθ = 1/r²

3. Singularity Handling:
   - r=0: Use r_safe = max(r, 1e-10) to avoid division by zero
   - Physically: ∂ψ/∂θ → 0 as r → 0 (regularity condition)

4. Volume Integration:
   - Jacobian: r (cylindrical coords: dV = r dr dθ dζ)
   - Rectangle rule: H = Σ E_ijk * r_i * dr * dθ * dζ
   - Error: O(h²) for smooth functions
