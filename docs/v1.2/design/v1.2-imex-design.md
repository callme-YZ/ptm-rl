## 1. Physics Formulation

### Equations

```
∂ψ/∂t = [ψ,φ] + η∇²ψ
∂ω/∂t = [ω,φ] + ν∇²ω
∇²φ = -ω
```

**Variables:**
- ψ: Poloidal magnetic flux
- ω: Vorticity
- φ: Stream function (velocity potential)
- η, ν: Resistivity, viscosity (realistic O(10⁻⁴))

**Toroidal geometry:** 
- Metric: g = 1/(R₀+r cosθ)²
- R₀=1.0, a=0.5

### Structure-Aware Components

**Hamiltonian part:** [ψ,φ], [ω,φ]
- Poisson bracket preserves energy (ideal limit)
- Arakawa scheme preserves identities

**Dissipative part:** η∇²ψ, ν∇²ω
- Realistic resistivity/viscosity
- Energy decay: dE/dt = -∫(η|J|² + ν|∇ω|²)dV

## 2. IMEX Integrator

### Splitting Strategy

**Explicit (RK2):** Hamiltonian terms
```python
ψ* = ψⁿ + Δt/2 * [ψⁿ,φⁿ]
ψ** = ψⁿ + Δt * [ψ*,φ*]
```

**Implicit (Backward Euler):** Dissipation
```python
(I - Δt*η*∇²)ψⁿ⁺¹ = ψ**
(I - Δt*ν*∇²)ωⁿ⁺¹ = ω**
```

### Advantages

✅ Unconditionally stable (implicit diffusion)  
✅ Structure-preserving (Arakawa bracket)  
✅ Allows realistic η,ν without Δt restriction  
✅ Moderate cost (1 Poisson solve + 2 implicit solves)

## 4.5 Island Width Diagnostic

### Method: O-point and X-point Detection

```python
def compute_island_width(psi, grid):
    """
    Compute magnetic island width from psi field.
    
    Returns
    -------
    width : float
        Island width in units of minor radius
        Returns 0 if no island detected
    """
    # Step 1: Find O-point (local extremum)
    grad_psi = np.gradient(psi)
    lapl_psi = laplacian(psi, grid)
    
    # O-point: ∇ψ=0 and ∇²ψ>0 (minimum)
    candidates_O = np.where(
        (np.abs(grad_psi[0]) < threshold) &
        (np.abs(grad_psi[1]) < threshold) &
        (lapl_psi > 0)
    )
    
    if len(candidates_O[0]) == 0:
        return 0.0  # No island
    
    # Step 2: Find X-point (saddle point)
    hessian = compute_hessian(psi, grid)
    det_H = hessian[0,0]*hessian[1,1] - hessian[0,1]**2
    
    # X-point: ∇ψ=0 and det(H)<0
    candidates_X = np.where(
        (np.abs(grad_psi[0]) < threshold) &
        (np.abs(grad_psi[1]) < threshold) &
        (det_H < 0)
    )
    
    if len(candidates_X[0]) == 0:
        # Fallback: use std(ψ) in resonant layer
        return np.std(psi[resonant_layer])
    
    # Step 3: Compute width
    O_point = (candidates_O[0][0], candidates_O[1][0])
    X_point = (candidates_X[0][0], candidates_X[1][0])
    
    width = 2 * np.linalg.norm(np.array(O_point) - np.array(X_point))
    width_normalized = width / grid.minor_radius
    
    return width_normalized
```

## 4.6 Baselines for Comparison

**Baseline 1: No Control**
```python
class NoControlBaseline:
    def act(self, obs):
        return np.zeros(2)  # [0, 0]
```

**Baseline 2: Fixed Boundary**
```python
class FixedBoundaryBaseline:
    def __init__(self, a0=0.5, a1=0.0):
        self.action = np.array([a0, a1])
    
    def act(self, obs):
        return self.action  # Heuristic constant
```

**Baseline 3: Random Policy**
```python
class RandomBaseline:
    def act(self, obs):
        return np.random.uniform(-1, 1, 2)
```

**Baseline 4: Simple Reactive**
```python
class ReactiveBaseline:
    def act(self, obs):
        # React to energy drift
        energy_drift = obs[-1]
        a0 = -0.5 * np.sign(energy_drift)  # Oppose drift
        a1 = 0.0
        return np.array([a0, a1])
```

**Success Criterion:**
```
RL suppression > max(baseline_1, baseline_2, baseline_3, baseline_4)
by at least 20%
```


```
2. Increase n_steps (2048 → 4096)

4. Decrease clip_range (0.2 → 0.1)
```
