## Key Findings

### 1. IMEX Fix Works ✅

**Before (with bug):**
```python
psi_new = psi_star + dt/2 * eta * J(psi_new)  # WRONG: anti-damping
```
- Result: NaN for η ≥ 1e-5

**After (fixed):**
```python
psi_new = psi_star - dt/2 * eta * J(psi_new)  # CORRECT: damping
```
- Result: Stable up to η = 1e-3 ✅

**Impact:** Achieved Phase 3 goal (large resistivity support)

---

### 2. Energy Dissipation Theory Mismatch 🔴

**Problem:** Numerical dH/dt ≠ Theory dH/dt

**Theory:** dH/dt = -η ∫ J² dV

**Observed:**
```
η = 1e-4, T = 0.2s
Step   0: dH/dt = -1.46e+00 (theory: -7.24e+07)  [100% error]
Step  40: dH/dt = -2.19e-03 (theory: -6.50e-02)  [97% error]
Step 160: dH/dt = -1.88e-03 (theory: -6.50e-02)  [97% error]
```

**Discrepancy:** Factor of ~30x (96-99% error)

**Hypotheses:**
1. **Hamiltonian definition:**
   - Code: `H = compute_hamiltonian(psi, omega, grid)` (passing vorticity)
   - Should: `H = compute_hamiltonian(psi, phi, grid)` (passing stream function)
   - Missing: φ from ∇²φ = ω

2. **Current density:**
   - `compute_current_density` returns J_φ (toroidal current)
   - Theory may use different J (e.g., ∂H/∂ψ)

3. **Theory derivation:**
   - Formula dH/dt = -η ∫ J² dV may be wrong
   - Need to derive from ∂H/∂t directly

**Impact:**
- ✅ Stability preserved (energy still decreases)
- ❌ Cannot claim "physics correct"
- 🔴 **BLOCKS v1.3 completion**

**Next Step:** Spawn theory review task (2-3 hours)

---

### 3. IMEX Performance Issue 🟡

**Observation:** 100 steps take ~3 minutes

**Cause:**
- Each step: 2× GMRES solve (implicit diffusion)
- Typical: 5-10 iterations per solve
- Grid: 64×64

**Impact:**
- Validation tests slow (convergence test ~30 min)
- Not suitable for RL training (needs 1000s of steps)

**Mitigation:**
- Use explicit solver for small η (η < 1e-5)
- Future: preconditioner or multigrid

**Priority:** 🟡 LOW (optimization, not correctness)
