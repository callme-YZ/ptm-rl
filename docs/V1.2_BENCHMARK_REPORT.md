### Phase 2: Symplectic Integration (2.5 hours)

**Objective:** Replace RK4 with structure-preserving integrator.

#### Implementation

**Method:** Störmer-Verlet operator splitting
```
ψ^(n+1/2) = ψ^n + (dt/2) [ψ, φ^n]
φ^(n+1/2) = solve Poisson(ω^n)
ω^(n+1) = ω^n + dt [ω, φ^(n+1/2)] + dissipation
ψ^(n+1) = ψ^(n+1/2) + (dt/2) [ψ, φ^(n+1)]
```

**Files:**
- `src/pytokmhd/integrators/symplectic.py` (new)

## Known Limitations (v1.2)

### 1. div_B Constraint Proxy ⚠️

**Issue:** `div_B_max` computed as max|∇²ψ|, not true ∇·B

**Impact:**
```
Laplacian proxy: O(10⁷)
True ∇·B: O(10⁻¹⁰) (expected)
Difference: 10¹⁷ ×
```

**Consequence:**
- Cannot use in reward (dominates ~80% even with w=1e-8)
- Set `w_constraint=0.0` for v1.2
- Observation includes proxy, but not used for control

**Workaround:** Option A (disable constraint in reward)

**Fix:** v1.3 will implement true ∇·B = ∇·(∇ψ × ∇φ/R)

**Physics Note:** True ∇·B ≡ 0 by construction (vector identity), proxy is numerical check only.

---

### 2. Cylindrical Equilibrium Profile ⚠️

**Issue:** Initial condition `ψ = r²(1-r/a)` is cylindrical, NOT toroidal equilibrium

**Impact:**
```
Energy transient: ~15%
Force balance: approximate
Grad-Shafranov: not satisfied
```

**Consequence:**
- Energy drift includes geometry mismatch
- Not purely numerical error
- Affects baseline interpretation

**Workaround:** Accept 15% transient as known physics

**Fix:** v1.3 Solovev analytical equilibrium

**Physics Note:** True toroidal equilibrium requires solving Grad-Shafranov equation.

---

### 3. Parameter Modulation Action Space ⚠️

**Issue:** Action space is (η, ν) multipliers, NOT physical actuators

**Impact:**
```
Action: [eta_mult, nu_mult] ∈ [0.5, 2.0]²
Physics: Modulates resistivity/viscosity
Reality: Not controllable in real tokamak
```

**Consequence:**
- Cannot transfer learned policy to experiment
- Framework validation only
- No realistic control demo

**Workaround:** v1.2 goal is framework, not deployable policy

**Fix:** v2.0 spatial current drive J_ext(r,θ) (6D action)

**Physics Note:** Real actuators are external coils (RMP), NBI, ECRH, etc.

---

### 4. Short Training Horizon

**Issue:** v1.2 tested with 100-step episodes

**Impact:**
```
Physical time: 100 × 1e-4 = 0.01 s
Tearing timescale: ~1-10 s
Coverage: <1% of full dynamics
```

**Consequence:**
- Cannot observe long-term island growth
- Limited policy generalization
- Fast transient control only

**Workaround:** v1.2 focuses on framework, not long-time physics

**Fix:** v1.3+ longer episodes with better equilibrium
