## Implementation Phases

### Phase 1: Hamiltonian Formulation ✅

**Completion:** 2026-03-19  
**Duration:** 1 day

#### Components Implemented

**1. Poisson Bracket Operator**
- File: `src/pytokmhd/operators/poisson_bracket.py`
- Canonical Poisson bracket: `[f, g] = (1/R²)(∂f/∂r ∂g/∂θ - ∂f/∂θ ∂g/∂r)`
- Verified mathematical properties:
  - Anti-symmetry: `[f, g] = -[g, f]` (precision 1e-14)
  - Linearity: `[af+bg, h] = a[f,h] + b[g,h]` (precision 1e-12)
  - Jacobi identity: residual 3e-3 (2nd order accurate)

**2. Hamiltonian Energy Functional**
- File: `src/pytokmhd/physics/hamiltonian.py`
- Total energy: `H[ψ, φ] = ∫ d³x [(1/2)|∇φ|² + (1/2)|∇ψ|²]`
- Kinetic energy (E×B flow): `K = ∫ (1/2)|∇φ|² dV`
- Magnetic energy (poloidal field): `U = ∫ (1/2)|∇ψ|² dV`
- Toroidal volume element: `d³x = r·R dr dθ dφ`, integrated over φ ∈ [0, 2π]

**3. Physics Module**
- File: `src/pytokmhd/physics/__init__.py`
- Exports: `compute_hamiltonian`, `kinetic_energy`, `magnetic_energy`

#### Validation

- Energy conservation verified to numerical precision (1e-12)
- Poisson bracket properties confirmed
- Grid convergence tested (32×32 to 128×128)

### Phase 3: IMEX Resistive Diffusion ✅

**Completion:** 2026-03-19  
**Duration:** 2 days (including today's energy budget debug)

#### Components Implemented

**1. Implicit Resistive Solver**
- File: `src/pytokmhd/solvers/implicit_resistive.py`
- Solves: `(I - dt·η·[J operator])ψ^(n+1) = ψ*`
- Method: GMRES with LinearOperator
- Current density: `J = Δ*ψ/R` (Grad-Shafranov operator)
- Convergence: GMRES tolerance 1e-8, typically converges in 5-20 iterations

**2. IMEX Time Stepper**
- File: `src/pytokmhd/solvers/hamiltonian_mhd_imex.py`
- Störmer-Verlet splitting:
  - Explicit: Poisson bracket (advection)
  - Implicit: Resistive diffusion `-η·J`
- Two half-steps per timestep for 2nd order accuracy

**3. Physics Integration**
- Resistive MHD evolution: `∂ψ/∂t = [ψ, φ] - η·J`
- Vorticity evolution: `∂ω/∂t = [ω, φ] - ν·∇²ω`
- Boundary conditions enforced implicitly

## Known Limitations

### 1. Energy Dissipation Error (20-45%)

**Issue:** Numerical dH/dt differs from theory by 20-45%

**Theory:** `dH/dt = -η ∫ J² dV`

**Observed:**
- Smooth IC: ~20% error
- Sharp IC: ~45% error

**Root Cause:**
- Implicit scheme + nonlinear Grad-Shafranov operator
- GMRES linearizes around current iterate
- Systematic bias in energy dissipation formula for nonlinear diffusion

**Impact:**
- ✅ Direction correct (energy decreases)
- ✅ Order of magnitude correct
- ⚠️ Quantitative mismatch for dissipation rate
- **Acceptable for v1.3 physics validation**

**Evidence:**
- Linear diffusion (Laplacian): 6% error (good!)
- Nonlinear GS operator: 20-45% error
- Error does not grow over time (stable bias)

**Mitigation:**
- Use smooth initial conditions (20% error)
- Document in physics validation reports
- Consider explicit scheme if exact dissipation needed (but loses stability)

### 2. Computational Cost

**IMEX solver slow:** ~0.5 sec/step vs 0.1 sec/step (symplectic only)

**Reason:**
- GMRES iteration for implicit solve
- Each iteration computes `J(ψ) = Δ*ψ/R` (expensive)

**Mitigation:**
- Good initial guess (current state) → fast convergence (5-20 iterations)
- Could use multigrid preconditioner (future work)

### 3. Grid Resolution Limits

**Current grid:** 32×64 adequate for validation, marginal for fine structures

**Recommendation:** 64×128 for production RL training

## API Specification

### Core Physics Functions

```python
from pytokmhd.physics import compute_hamiltonian, compute_current_density
from pytokmhd.operators import poisson_bracket
from pytokmhd.solvers import HamiltonianMHDIMEX

# Compute energy
H = compute_hamiltonian(psi, phi, grid)

# Compute Poisson bracket
bracket = poisson_bracket(f, g, grid)

# Create IMEX solver
solver = HamiltonianMHDIMEX(
    grid=grid,
    dt=1e-4,
    eta=1e-4,  # resistivity
    nu=0.0,    # viscosity
    use_imex=True
)

# Evolve one timestep
psi_new, omega_new = solver.step(psi, omega)
```

### Environment Integration

```python
from pytokmhd.rl import ObservationWrapper

# Create observation from MHD state
obs_wrapper = ObservationWrapper(solver, grid)
observation = obs_wrapper.get_observation(psi, omega)

# observation is (19,) array:
#   - Energy metrics (3)
#   - div_B constraint (1)
#   - Fourier modes (4)
#   - Gradients (4)
#   - Other physics (7)
```
