
**Analysis:**
- Numerical dissipation is **~30x smaller** than theory predicts
- Theory: dH/dt = -η ∫ J² dV  (where J = Δ*ψ/(μ₀R))
- Possible issues:
  1. Wrong J definition (using J_φ, need different form?)
  2. Hamiltonian should use φ not ω: H(ψ, φ) vs H(ψ, ω)
  3. Theory formula itself needs re-derivation

**Plot:** `results/test_imex_energy_budget.png`

**Conclusion:**
- ✅ Stability preserved

## Key Findings

### 1. IMEX Sign Fix Works ✅

**Before:**
```python
psi_new = psi_star + dt/2 * eta * J(psi_new)  # WRONG!
```

**After:**
```python
psi_new = psi_star - dt/2 * eta * J(psi_new)  # CORRECT damping
```

**Impact:**
- Resistive term now damps (negative sign)
- Stable for η up to 1e-3 (was failing at 1e-5 before)

---

### 2. Energy Dissipation Mismatch ⚠️

**Expected:** dH/dt ≈ -η ∫ J² dV

**Observed:** dH/dt ≈ -0.03 × η ∫ J² dV  (factor of 30 discrepancy)

**Hypothesis:**
1. **Hamiltonian definition issue:**
   - Current code: `H = compute_hamiltonian(psi, omega, grid)`
   - Correct: `H = compute_hamiltonian(psi, phi, grid)` where φ from ∇²φ = ω
   - Need to solve Poisson equation each time (slow but correct)

2. **Current density definition:**
   - `compute_current_density` returns J_φ (toroidal current)
   - Theory uses different J? Check ∂H/∂ψ relation

3. **Volume integral:**
   - Current: `dV = R * dr * dθ`
   - Correct? Need to check Jacobian

**Next step:** Spawn theory review task

---

### 3. Performance Issue

**IMEX solver is slow:**
- 100 steps: ~3 minutes
- 800 steps: ~30 minutes (est.)

**Reason:**
- Each step: 2× implicit solve (GMRES on 64×64 grid)
- Convergence: 5-10 iterations per solve

**Mitigation:**
- Use preconditioner (future)
- Reduce grid for tests (trade-off: accuracy)
- Explicit fallback for small η
