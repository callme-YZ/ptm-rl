## Blocker: Axial Singularity

### Problem

```python
# src/pytokmhd/physics/force_balance.py:135
Delta_star_psi = d2psi_dr2 + d2psi_dtheta2 / r_grid**2 + ...
```

At r=0 (axis), `1/r²` term diverges:
- `r_grid[0] ≈ 0.045` → `r²  ≈ 0.002`
- `d²ψ/dθ² / r²` → O(10³)
- `J_phi` → O(10⁶) (observed)
- Time evolution → NaN after ~1 step

### Evidence

```
Initial IC check:
  psi:   [0.000e+00, 1.333e-02], no NaN ✅
  omega: [-2.617e+00, 5.101e+00], no NaN ✅
  J:     [-4.061e+06, 8.888e+05], no NaN but HUGE ❌

After 1 step:
  H = nan, drift = +nan% ❌

Warnings:
  RuntimeWarning: overflow encountered in divide
  RuntimeWarning: invalid value encountered in subtract
```

### Fix Required

**Option 1: L'Hôpital Rule at Axis**
```python
if i == 0:  # r = 0
    # Use regularity: lim (d²ψ/dθ²)/r² = finite
    Delta_star_psi[0, :] = d2psi_dr2[0, :] + ... (special formula)
else:
    Delta_star_psi[i, :] = d2psi_dr2[i, :] + d2psi_dtheta2[i, :] / r_grid[i, :]**2
```

**Option 2: Avoid r=0**
- Start radial grid at r_min = 0.1·a
- Modify `ToroidalGrid` to support `r_min` parameter

**Option 3: Axisymmetry Enforcement**
- At r=0, enforce ψ(0, θ) = constant → dψ/dθ = 0 → d²ψ/dθ² = 0
- No division by zero

**Recommendation:** Option 3 (physically correct + simple)

**Estimate:** 3-4 hours to implement + test

**Assumption:**

**Impact:**

**Learning:**

**Choice:**  
Use simple ψ = r²(1-r/a) for all tests

**Reality:**  
Not a toroidal equilibrium → force balance fails

**Impact:**  
1/6 property test fails (expected, but still a fail)

**Learning:**  
Exact equilibrium (Solovev) needed for strict validation
