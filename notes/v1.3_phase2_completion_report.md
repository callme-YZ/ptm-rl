## Mathematical Formulation

### Pressure Profile
```
P(ψ) = P₀(1 - ψ/ψ_edge)^α    for ψ < ψ_edge
     = 0                      for ψ ≥ ψ_edge

dP/dψ = -α·P₀/ψ_edge · (1 - ψ/ψ_edge)^(α-1)

∇P = (dP/dψ)·∇ψ
```

### Force Balance
```
J×B = ∇P

Jφ = (1/μ₀R)Δ*ψ

Δ*ψ = ∂²ψ/∂r² + (1/r²)∂²ψ/∂θ² + (cos θ/R)(∂ψ/∂r)

(J×B)ᵣ = Jφ·(∂ψ/∂r)
(J×B)θ = Jφ·(1/r)·(∂ψ/∂θ)
```

### Vorticity Source
```
∂ω/∂t = [ω, H] + S_P + dissipation

S_P = (1/R²)(dP/dψ)·Δ*ψ
```
